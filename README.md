## Installation

Create an `.env` file in the root directory with the following variables:

```bash
DISCORD_TOKEN=
POSTGRES_URL=
GUILD_ID=
GENERAL_CHANNEL_ID=
BOT_CHANNEL_ID=
AWS_ACCESS_KEY=
AWS_SECRET_ACCESS_KEY=
PREFIX=
LAVALINK_URI=
LAVALINK_PASSWORD=
HEALTHCHECK_URL=
X-API-KEY=
```

Run the bot:

```bash
./run.sh
```
