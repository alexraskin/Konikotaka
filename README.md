## Installation

Creat an `.env` file:
```bash
DISCORD_TOKEN=
MYSQL_URL=
OPENAI_API_KEY=
GUILD_ID=
GENERAL_CHANNEL_ID=
BOT_CHANNEL_ID=
```

Run the bot:
```bash
./run.sh
```
