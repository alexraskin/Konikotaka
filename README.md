## Description
OldWiseManBot: A mystical Discord companion, weaving enchanting wisdom and magic into your servers for a magical experience

## Installation

Creat an `.env` file: 
```
DISCORD_TOKEN=""
MYSQL_URL=""
GUILD_ID=
GENERAL_CHANNEL_ID=
BOT_CHANNEL_ID=
```

Run the bot:
```bash
./run.sh
```
